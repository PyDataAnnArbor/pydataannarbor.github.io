Building Software for Biology Research: Into the Cells

written by Sean Law and Benjamin Zaitlen on 2020-03-11

New imaging techniques are enabling biomedical discovery on a whole new scale. This talk explores how the Chan Zuckerberg Initiative is building open source tools in python for processing the growing amount of data needed for biomedical research.

Shannon is a senior software engineer at the Chan Zuckerberg Initiative. She writes software to help biologists process, analyze, and visualize their data.
